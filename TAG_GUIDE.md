# 古文単語 タグ定義・判断基準ガイド

330語に付与するタグの「定義」と「付与の判断基準」をまとめたリファレンス。
新語を足すとき・分類に迷ったときはこの基準に従う。

> この体系はレビューで確定した最新版。`build.py` の `TL`（テーマ定義）・`TH`（語→テーマ割り当て）・フラグ各 set に反映する（実装の仕組みは [TAGGING.md](TAGGING.md) を参照）。

---

## 0. 基本の考え方（全タグ共通）

1. **2つの面（facet）で捉える**
   - **意味テーマ** … その語が「どんな意味の世界に属するか」＝〔領域8種〕＋〔評価2種〕。
   - **覚え方タグ** … 学習上の注意点・覚え方のヒント（10種）。
2. **多ラベル（重複可）** … 排反ではなく、**該当するタグはすべて付ける**。
   例：つれなし＝「恋・男女」＋「マイナスの評価・心情」、あはれ＝「情趣」＋「プラスの評価・心情」＋「漠然系」。
3. **中立・非該当は無印（omission）** … 当てはまる分野が無ければ付けない。「その他」の箱は作らない（弱い手がかりになるため）。
4. **機能語は品詞で足りる** … 指示・程度・呼応などの副詞や連語は、意味の分野を持たないので**テーマ無印でよい**（品詞フィルタで拾える）。
5. **例文の解答・訳は変えない** … タグは分類のためのメタ情報。語義・例文には手を入れない。

## 1. 意味テーマ（10種）

### 1-A. 領域タグ（8種）― 「その語はどんな世界の語か」

| タグ | 定義 | 付与基準 | 例 |
|---|---|---|---|
| **恋・男女** | 恋愛・結婚・男女の仲 | 求愛・結婚・逢瀬・通い・心変わり・男女の親密を表す語 | あふ／しのぶ／みゆ（結婚）／すむ（通う）／ちぎり／みそかなり／よのなか／つれなし（薄情） |
| **病・死・別れ** | 病気・死・別離・弔い | 病む・体調・死ぬ・先立たれる・見舞い・弔問を表す語 | うす／なやむ／わづらふ／おこたる（治る）／おくる（先立たれる）／あつし／ここち／とぶらふ |
| **仏道・信仰** | 仏教・信仰・出家・無常 | 勤行・祈り・出家・宿縁・霊験など仏教世界の語 | ねんず／おこなふ／ほい／しるし（霊験）／ちぎり（宿縁） |
| **宮廷・身分** | 朝廷・身分・官職・皇室 | 天皇・宮中・身分の高下・政・皇室の動作を表す語 | おほやけ／うち／うへ／きは／しな／よし（身分）／あやし（身分低）／まつりごつ／行幸／そうす／けいす |
| **情趣** | あはれ・をかし的な美的情感の世界 | しみじみとした趣・風流・興趣・風情を表す語（**「美・優雅」は情趣に入れず、プラスの評価・心情へ**） | をかし／あはれ／おもしろし／なさけ（風流心）／あそぶ（管絃）／いろ（風情）／こころ（風流心）／ゆゑ（風情） |
| **時・時間** | 時刻・時期・時の経過 | 時・時分・早朝・月末・すぐに・以前・一晩中など時を表す語（※季節・自然の語は実質無いのでラベルは「時・時間」） | としごろ／つとめて／つごもり／やがて／すなはち／いつしか／よもすがら／またのひ |
| **ことば・学問** | 言葉・文・学問・虚実 | 手紙・物語・漢学・才・言い誤り・嘘など言葉/学問に関わる語 | ものがたり／ふみ／ざえ／そらごと／せうそこ／うちいづ／ひがごと |
| **思考・評判** | 内面の思考・認識と、世間の評判・うわさ | 思う・気づく・想像する（思考）／評判・うわさ・世に知られる（評判） | おどろく／おぼゆ／きこゆ／ののしる（評判）／おとにきく／おもひやる／おぼえ（評判） |

**領域タグの注意**
- 1語が複数領域にまたがれば全部付ける（例：みゆ＝恋・男女＋思考・評判、なさけ＝情趣＋恋・男女）。
- 純粋に価値中立の具体語（かたち・はらから・ゐる 等）は領域なし＝無印でよい。

### 1-B. 評価タグ（2種）― 「プラスかマイナスか」

| タグ | 定義 | 例 |
|---|---|---|
| **プラスの評価・心情** | 良い評価、または肯定的な感情 | めでたし／うつくし／かしこし／なつかし／らうたげなり／かたじけなし／たのし（裕福） |
| **マイナスの評価・心情** | 悪い評価、または否定的な感情 | うし／わびし／あさまし／むつかし／つれなし（冷淡）／うたて／をこがまし |

**評価タグの判断基準**
- 付けるのは **「人・物・事の良し悪しを特徴づける語」または「話者の感情を表す語」**。
- 単なる記述（状態・容姿・動作）で良し悪しを含まない語には付けない（例：おとなし＝年配 は付けない／分別がある の面を評価とみて付けるかは要判断）。
- **旧「よい評価／わるい評価／心情・つらさ／心情・いとしさ／不安・気がかり／美・優雅」はすべてこの2タグに統合**。
- **両方に当てはまる語（＋にも−にもなる）は両方付ける** → 自動的に「プラス・マイナス両義」フラグになる（§2参照）。例：いみじ・ゆゆし・さかし・しどけなし・こよなし・おぼろけなり・いとほし。

## 2. 覚え方タグ（10種）

| タグ | 定義 | 付与方法・基準 | 例 |
|---|---|---|---|
| **現代語とギャップ大** | 現代語と意味が**大きく**違い、現代感覚が通じない | 手動。同形で意味が別物の語 | しる（領有）／あくがる／おこたる（治る）／やがて（すぐ）／せめて（無理に）／ときめく |
| **現代語と少しズレ** | 現代語と意味が**少し**ずれる | 手動（ギャップ大と排他） | たのむ（あてにする）／さはる（妨げ）／なやむ（病む）／わざと（わざわざ）／ふるさと（古都） |
| **多義語** | 別々の語義を多く持つ | **自動**：統合後の意味が4つ以上 | おぼゆ／ほど／たより／あやし |
| **コアで覚える** | 一つの中心イメージから多義が派生。中心義で覚えると効率的 | 手動 | おどろく（ハッと）／おぼゆ／みゆ／いみじ |
| **漠然系** | 意味の輪郭がぼやけ、現代語一語に訳しにくい。文脈で具体化する | 手動（§3の判定テスト） | ほど／けしき／よし／ゆゑ／やう／さた／ものす／わたる／あはれ／をかし |
| **プラス・マイナス両義** | 同語が文脈で＋にも−にもなる（入試の罠） | **自動**：プラス評価・心情とマイナス評価・心情の**両テーマを持つ語**から導出 | いみじ／ゆゆし／さかし／しどけなし／こよなし／おぼろけなり／いとほし |
| **呼応の副詞** | 下に打消・禁止・推量・願望・疑問反語などを呼び起こす陳述副詞 | 手動 | え〜（打消）／な〜そ（禁止）／さらに〜（打消）／よも〜じ／いかで／など・なでふ（疑問反語）／かまへて |
| **「〜なし」型** | 「〜なし」の形の形容詞（現代語の「無い」と違う意味に注意） | **自動**：品詞＝形容詞かつ見出しが「なし」で終わる | はかなし／こころもとなし／おぼつかなし／さがなし |
| **漢語サ変** | 漢語＋「す」のサ変動詞 | 手動 | ねんず／きようず／こうず／そうす／けいす |
| **敬語要注意** | 敬語の種類・本動詞/補助動詞の識別が問われる語 | 手動 | たまふ（四段尊敬↔下二丁寧＝159/160）／はべり・さぶらふ（丁寧↔謙譲）／きこゆ（動詞↔申し上げる）／たてまつる／めす／まゐる |

## 3. 迷いやすい区別と判断のコツ

### (1) 漠然系 か 多義語 か
- **多義語**：語義が「思われる／似る／思い出す」のように**別個の意味として現代語一語で言い切れる**。→ おぼゆ・なさけ・きこゆ。
- **漠然系**：**現代語一語で言い切れず「〜という感じ」としか言えない**、または指す範囲が広く文脈依存で決まる。
- **判定テスト**：「その語を現代語一語で言い切れるか？」 言い切れない＝漠然系／言い切れる別義が並ぶ＝多義語。
- 両方に当てはまる語（ほど 等）は**両方付ける**。

### (2) 漠然系の2タイプ
- **(a) 広義・投射型**：一つの広い概念が時・空間・程度・身分…と多方面に投射する。例：ほど・けしき・よし・ゆゑ・やう・さた・ひま・かぎり・ものす・わたる・ありく。
- **(b) 情感・印象型**：対応する現代語を持たない核心的情感語に**限定**。例：**あはれ・をかし**。
  - 「退屈／物寂しい」のように**名指せる濃淡に訳し分けられる語（つれづれ・ながむ）は漠然系にしない**（評価・心情タグのみ）。

### (3) 「両義」は「両テーマ」と一致させる
- 「プラス・マイナス両義」フラグ ⇔ その語が**プラス評価・心情とマイナス評価・心情の両方を持つ**。
- 別リストで管理せず、両テーマの有無から**自動導出**する（食い違いが原理的に起きない）。
- 語義が片側だけの語（あさまし＝マイナスのみ）には付けない。

### (4) 情趣 と 美・優雅 と プラス
- **美・優雅（視覚的な美しさ・優美）** … 領域タグにはせず、**プラスの評価・心情**に入れる。例：にほふ・きよらなり・なまめかし・うるはし。
- **情趣（あはれ・をかし的な情感）** … 独立の領域タグ。肯定的なら **プラスの評価・心情も併記**（例：あはれ＝情趣＋プラス）。

### (5) 現代語ギャップ大 と 少しズレ
- **ギャップ大**：現代の意味では文意を取り違える（しる＝領有、やがて＝すぐ）。
- **少しズレ**：現代の意味から推測はできるが精度が落ちる（なやむ＝病む、わざと＝わざわざ）。
- 目安であり境界は緩い。優先度づけ（ギャップ大＝まっさらに覚える）に使う。

### (6) 無印（omission）でよい語
- **機能語**（副詞・指示・呼応・連語）：品詞フィルタで拾えるのでテーマ無印。例：いと・げに・かく・さ・さて・いざたまへ。
- **価値中立の具体名詞・動詞**：かたち・かげ・はらから・ゐる・まうく・わななく 等。
- **広義の抽象語**：テーマではなく「漠然系」フラグで拾う（よし・ゆゑ・やう・さた 等）。
