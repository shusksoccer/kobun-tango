# タグ分類・意味統合・関連語ロジック

すべて `build.py` 内で完結。番号（`no`）ベースで管理しているので、ソースの番号が変わらない限りマップは有効。

## 1. 重要度 `imp`（番号レンジ）

`importance(no)`:

| レンジ | 値 |
|---|---|
| 1–50 | 最必修 |
| 51–150 | 必修 |
| 151–180 | 必修敬語（敬語30語のブロック）|
| 181–280 | 重要 |
| 281–330 | 応用 |

## 2. 敬語種別 `keigo`

`KEIGO`（dict: no → ['尊敬'|'謙譲'|'丁寧', ...]）を 151–180 に手動付与。`たてまつる(164)`・`まゐる(169)` など尊敬/謙譲両用は両方を持つ。

## 3. 意味テーマ `themes`（13種）

`TL`（コード→ラベル）と `TH`（no → コード配列）で手動付与。13種:
恋・男女 / 病・死・別れ / 仏道・信仰 / 宮廷・身分 / 美・優雅・情趣 / よい評価 / わるい評価 / 心情・つらさ / 心情・いとしさ / 不安・気がかり / 思考・うわさ / 時・季節・自然 / ことば・学問。
内容語のみ付与（機能語的な副詞などは無印で可）。

## 4. 覚え方フラグ `flags`（9種）

| フラグ | 付与方法 |
|---|---|
| 現代語とギャップ大 | `GAP_BIG`（手動 set）|
| 現代語と少しズレ | `GAP_SMALL`（手動 set。GAP_BIG と排他）|
| 多義語 | **自動**: 統合後の意味数 ≥ 4 |
| コアで覚える | `CORE`（手動 set。1コアから派生する語）|
| プラス/マイナス両義 | `POLAR`（手動 set: いみじ/ゆゆし/あさまし/なかなか）|
| 呼応の副詞 | **自動**: 見出しに `～` を含む |
| 「〜なし」型 | **自動**: 品詞=形容詞 かつ 見出しが `なし` で終わる |
| 漢語サ変 | `KANGO`（手動 set）|
| 敬語要注意 | `KEIGO_CARE`（手動 set: 尊敬/謙譲両用など）|

新語追加・意味変更時、`多義語` は意味統合の結果で自動的に増減する点に注意。

## 5. 意味の統合（重複の解消）＋終止形化 ★重要

**背景**: 元データの「意味」列は例文ごとの解答（活用形・重複あり）。そのままだと「例文の数＝意味の数」になり重複が多い。そこで単語ごとに同義の解答を1つに統合し、表示用ラベルを終止形（言い切り）にする。**例文の解答 `imi` は不変**（例文に合う形のまま）。

手順（assemble ループ内）:
1. 例文解答の重複しない一覧を作る。
2. `cluster_meanings()` で「**同一語の活用語尾の違い・重複だけ**」を統合。判定 `_same()`:
   - 重複（`すき間すき間`→`すき間`）は `_collapse()` で畳む。
   - 共通接頭辞があり、両者の差分（残り）が `ALLOWED`（純粋な活用語尾の閉集合）に入るときのみ統合。
   - これにより「かわいそう／かわいい」「とても／とても悲しい」のような**別義の誤統合を防止**（差分が活用語尾でないため非統合）。
3. `canon_label()` で代表ラベルを選択（終止形寄りの語尾を優先、`OVERRIDE` で少数の名詞化曖昧語などを補正）。
4. `lemmatize()`（**fugashi+unidic-lite**）で末尾の活用語を終止形化。末尾形態素の `orthBase`（語彙素の基本形）に置換。
   - 例: `見られ→見られる`、`目を覚まし→目を覚ます`、`裕福な→裕福だ`、`滑稽な→滑稽だ`。
   - 名詞・副詞の訳語（祈り／やはり 等）は活用語でないため不変。
   - fugashi未導入時は恒等関数（終止形化されないが動作はする）。
5. 終止形化後に同一化したラベルを再統合し、各例文に統合後インデックス `mi` を付与。

**検証手段**: POS付き監査（末尾形態素が動詞/形容詞/形容動詞/助動詞かつ surface≠基本形 のラベル）が0件であること。新語追加後は同等チェックを推奨。

## 6. 関連語（類義・対義・混同）

番号ベースのクラスタ/ペアを手動定義し、対称化して各語に展開。
- `SYN`（類義クラスタ）/ `CONF`（混同クラスタ）/ `ANT`（対義ペア）。各要素は `(no, この語側の訳ラベル)`。
- `add_clusters()` がクラスタ内の全ペアを相互リンク、`add_ant()` が対義を双方向に追加。`(type,no)` で重複排除。
- 表示は `sense`（訳ラベル）でグルーピング。みゆ(7) なら「結婚する→あふ・すむ」「思われる→おぼゆ」のように**訳ごとに関連語が変わる**。
- 同形異義（よし16/122、しるし110/292、たまふ尊敬159/丁寧160、きこゆ60/156）も混同で相互リンク。
- 現状 92語に付与。最必修・必修・敬語の主要クラスタ中心で、**拡充の余地あり**（→ NEXT_STEPS）。
