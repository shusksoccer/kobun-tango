# データスキーマ（build/data_payload.json）

`build.py` の出力。`build_site.py` の入力。UTF-8 JSON。トップレベルは `{ "words": [...], "meta": {...} }`。

## words[]（330件、`no` 昇順）

各単語オブジェクト:

| キー | 型 | 説明 |
|---|---|---|
| `no` | int | 単語番号（1–330、欠番なし）。**全対応の主キー**（解説・関連語・ジャンプ先）。|
| `word` | string | 見出し語（歴史的仮名遣い）。例: `みゆ`、`のたまふ・のたまはす`、`え～打消`。|
| `pos` | string | 品詞。`動詞/形容詞/形容動詞/名詞/副詞/連体詞・連語` のいずれか。|
| `imp` | string | 重要度。`最必修/必修/必修敬語/重要/応用`。番号レンジで決定（→ TAGGING）。|
| `keigo` | string[] | 敬語種別。`尊敬/謙譲/丁寧` の0個以上（敬語語のみ）。|
| `meanings` | string[] | **意味一覧（統合済み・終止形）**。表示用。活用語は言い切り。|
| `themes` | string[] | 意味テーマ（10種のうち0個以上・多ラベル）。|
| `flags` | string[] | 覚え方フラグ（10種のうち0個以上）。|
| `kat` | string | 活用情報。動詞＝行・活用型（例 `ヤ行下二段`、両用は `／` 併記）、形容詞＝`ク活用/シク活用`、形容動詞＝`ナリ活用`、その他は空。|
| `related` | object[] | 関連語（類義・対義・混同）。下記。|
| `note` | string | 解説本文（explanations.md 由来。ポップアップに表示）。|
| `kanji` | string | 見出しの漢字表記（解説の `（…）` 由来。270語に有り、無い語は空文字）。|
| `examples` | object[] | 例文（下記）。|

### examples[]

各例文（＝元データの1行）:

| キー | 型 | 説明 |
|---|---|---|
| `imi` | string | **この例文の解答**（空欄に入る語＝活用形のまま、終止形化しない）。|
| `mi` | int | この例文が属する統合後の意味番号（1始まり。`meanings` のインデックス+1）。テストの番号表示・意味との対応に使用。|
| `ko` | string | 古文の全文（出典を除去済み）。|
| `koBlank` | string | 古文の空欄版（解答箇所が `〔　　　〕`）。|
| `koU` | string | `ko` の解答箇所を `\x01…\x02`（U+0001/U+0002）で囲んだもの。表示時に `<u class="mk">…</u>` へ変換（下線）。|
| `yk` | string | 現代語訳の全文（解答箇所が `〔答え〕`）。|
| `ykBlank` | string | 現代語訳の空欄版（`〔　　　〕`）。|
| `src` | string | 出典（例: `源氏物語`）。無い場合は空。|

### related[]

| キー | 型 | 説明 |
|---|---|---|
| `type` | string | `類義` / `対義` / `混同`。|
| `sense` | string | **この語側**の、どの訳に対する関連かを示すラベル（訳ごとの括り）。|
| `no` | int | 関連先の単語番号（クリックで辞典の該当カードへ）。|
| `word` | string | 関連先の見出し語。|
| `gloss` | string | 関連先側の訳ラベル。|

例: みゆ(7) の `{type:'類義', sense:'結婚する', no:8, word:'あふ', gloss:'結婚する'}` = 「『結婚する』の意味で あふ が類義」。

## meta

| キー | 型 | 説明 |
|---|---|---|
| `imp` | string[] | 重要度の表示順 `['最必修','必修','必修敬語','重要','応用']`。|
| `pos` | string[] | 品詞の表示・ソート順（6種）。|
| `keigo` | string[] | `['尊敬','謙譲','丁寧']`。|
| `theme` | string[] | テーマ13種の順序。|
| `flag` | string[] | フラグ9種の順序。|
| `total` | int | 総語数（330）。|

フィルタUIの選択肢は全て `meta` から動的生成。色は `build_site.py` の `IMP_C / KEIGO_C / THEME_C / FLAG_C` で定義。
