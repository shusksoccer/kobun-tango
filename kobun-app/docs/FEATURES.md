# UI仕様（dist/index.html・dist/quiz.html）

モバイル前提（スマホ主用途）。**PWA対応**（manifest＋Service Worker。ホーム画面に追加可・オフライン動作・フォントも初回訪問後にキャッシュ）。フォントは Google Fonts（Shippori Mincho B1 / Zen Kaku Gothic New）、未取得時は端末の明朝・ゴシックにフォールバック。色は伝統色ベースで観点ごとに固定。

## 辞典 index.html

- **ヘッダ**: タイトル＋総語数（`#subtitle` に `meta.total` を動的表示）、暗記テストへのリンク。
- **学習レポート**（`レポート` ボタン→モーダル）: 語・意味単位の既習数、重要度帯別／テーマ別の既習率バー、あいまい（弱点）語と復習期限切れ語の一覧（タップで該当カードへ）。
- **検索**: 単語・漢字・意味・テーマ・フラグを対象に部分一致。仮名は正規化して照合（カタカナ→ひらがな、ゐ・ゑ・を・ぢ・づ→現代仮名。「おかし」で「をかし」がヒット）。入力は150msデバウンス。
- **絞り込み**: 観点別チップ（重要度／品詞／敬語／意味テーマ／覚え方）。観点内はOR、観点間はAND。`meta` から自動生成。
- **並べ替え**: 番号順／五十音順／品詞順。
- **表示フィルタ**: すべて／未習のみ／既習のみ（既習＝その語の**全意味**を覚えた語。`localStorage` の意味単位の既習集合で判定）。
- **カード**:
  - 左の色帯＝重要度。左上の○は「**全意味の一括トグル**」（空＝未習／数字＝一部既習／✓＝全部既習）。全意味既習で薄表示化＋既習カウント更新。
  - 見出し語（明朝・大）＋漢字〔　〕＋重要度ラベル。品詞の隣に**活用情報**（動詞＝行・活用型／形容詞＝ク・シク／形容動詞＝ナリ）。
  - **タグは見出し直下の1行**にまとめて表示（品詞・敬語・テーマ・覚え方）。タグタップでその観点に絞り込み。
  - 意味（番号つき、終止形）。**各意味の番号○をタップで「その意味を覚えた」チェック**。意味単位で `koten_known`（`"番号:意味番号"`）に保存し、テストと同期。
  - **解説プレビュー欄**（右／スマホは下に全幅）: 冒頭数行が読め、タップで全文ポップアップ。
  - 折りたたみ: 「例文 n」「関連・混同 n」。例文は古文（空欄＋出典**（成立時代つき）**＋答え）＋訳。関連語は訳ごとにグルーピングしクリックで該当語へジャンプ。
- **解説ポップアップ**（`#noteModal`）: 下からのシート（PCは中央）。見出し・漢字・重要度・品詞・敬語・意味＋解説本文。背景／×／Escで閉じる。

## 暗記テスト quiz.html

- **設定**: 出題の向き（単語→意味／意味→単語／**四択**）、絞り込み（重要度・品詞・テーマ・覚え方）、問題数（10/20/30/50/すべて）、順番（ランダム／番号順）。
- **順方向（単語→意味）**:
  - 見出し語＋**古文の例文全文**（問われている語に**下線** `mk`）。
  - **訳ヒントはタップで開閉**（`ヒント（訳）を見る`。訳は空欄版）。
  - 「答えを見る」→ 意味（番号つき・終止形）＋訳全文。
- **逆引き（意味→単語）**:
  - 番号つきの**意味一覧**が問題文。
  - **例文を全件**表示（意味の数だけ）。各例文＝古文（空欄）＋訳。**下線は訳側のみ**（古文の空欄には下線なし）。
  - 例文一覧は**高さ固定でスクロール**（`.qexlist`、約40vh）。
  - 「答えを見る」→ 古語＋全例文（古語に下線・訳つき）。
- **共通**: 答え合わせの「辞典で詳しく確認 →」で `index.html#w{no}` へジャンプ。答え表示には**意味ごとのチェックリスト**が出て、覚えた意味に個別チェックできる（辞典と同期）。「覚えた／あいまい」で進行し、「覚えた」は出題された意味（単語→意味）／全意味（意味→単語）を既習にする（チェック日時も記録）。「あいまい」は `koten_weak` に永続化され、「覚えた」で解除。終了画面で「もう一度／あいまい復習／設定へ」。
- **四択テスト**: 語＋例文を見て正しい意味を4択から選ぶ。誤答肢は混同語→対義語→同テーマ・同品詞の意味から自動生成（自語の語義と重なる候補は除外）。正解＝出題意味を既習化・弱点解除、誤答＝あいまい（弱点）に記録。
- **復習キュー（SRS-lite）**: 設定画面に「あいまいにした語＋覚えたチェックから30日たった語」の復習キューを表示し、まとめて出題できる。

## 既知のUI制約・注意

- `localStorage` は try/catch でガード。ローカルで開けば保存される。サンドボックス埋め込みプレビューでは保存されないが動作はする。
- 生成HTMLにデータを内蔵（各約1MB）。
- 解説は辞典のみ（テストには出さない方針）。
- 下線は `koU` の `\x01/\x02` を `<u class="mk">` へ、空欄は `blankify()` が `〔　　　〕`→`<span class="blank">` へ変換。`） ］` での閉じ括弧ゆれにも対応済み。
