## 触るときのヒント
- まず該当が **build.py（データ/タグ）** か **build_site.py（表示/操作）** かを切り分ける。
- 変更後は必ず `python3 build.py && python3 build_site.py && node test/test.js`。
- build_site.py は HTML/CSS/JS を文字列で保持。**f-string禁止**（`%…%` の `.replace()`）。JSの正規表現・テンプレートリテラルの `${}` をそのまま含むので編集時はクオートに注意。
